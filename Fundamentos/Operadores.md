# Operadores #

## Operadores Aritméticos ##

* Adição:        +
* Subtração:     -
* Multiplicação: *
* Divisão:       /
* Módulo:        %
* Exponenciação: Math.Pow(base, expoente)




```
  + , - , * , / , % , Math.Pow(base, expoente)
```

## Operadores Relacionais ##

O resultado de uma operação relacional é 

```
  + , - , * , / , % , Math.Pow(base, expoente)
```

# Tipos em C# #

* Tipos Valor 

  * Tipos Simples:
  
    * Inteiro com sinal:
      * sbyte  (1 byte)
      * short  (2 bytes)
      * int    (4 bytes)
      * long   (8 bytes)
      
    * Inteiro sem sinal:
      * byte   (1 byte)
      * ushort (2 bytes)
      * uint   (4 bytes)
      * ulong  (8 bytes)  
      
    * Ponto flutuante: em C#, todo número de ponto flutuante é um double, para que seja float, é necessário inserir um *f* após o valor (exemplo: float raio = 5.18f;)
      * float  (4 bytes)
      * double (8 bytes) 
      
    * Decimal de alta precisão:
      * decimal (16 bytes)
      
    * Caractere unicode: delimitado por ' '
      * char (UTF-16)
         
    * Booleano: true ou false
      * bool 
  
  * struct
  * enum
